# 🚀 IntervueX AI

IntervueX AI is an AI-powered mock interview platform built to help students and job seekers prepare for technical and behavioral interviews.

## ✨ Features

### 📄 Resume Upload
- Upload resume in PDF format
- Resume management dashboard
- Ready for AI-powered analysis

### 📊 Resume Analysis
- ATS Score
- Matched Skills
- Missing Skills
- AI Feedback & Suggestions

### 🎤 Mock Interview
- Interactive interview experience
- Technical & behavioral questions
- Multiple interview rounds

### 📈 Performance Analytics
- Readiness Score
- Interview Statistics
- Progress Tracking
- Performance Overview

### 🏆 Leaderboard
- Global ranking system
- Score comparison
- Competitive learning experience

### 👤 User Profile
- Personalized dashboard
- Interview history
- Activity tracking

### 🔐 Authentication
- Clerk Authentication Integration
- User Sign Up & Login

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/intervuex-ai.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
